Tuma Board Games

What is this website going to be

This should be a first attempt to make the website. Graphics and structure will be neglected now. The main goals I'll try to tackle are:

  1. Making JS classes with good APIs capturing simple games logic.
  2. Adding very simple login and room creation mechanism.
  3. Experiment a little with JS interactions and animations and make an ugly but playable version of Chinese Checkers.

What is the long-term goal

A website hosting multiple free boardgames, accessible on any platform (mobile, tablet, laptop) and as many browser as possibile (chrome and firefox are the bare minimum).

This website Zen

  1. Ugly is better than broken.
  2. Simple is better than copyrighted.
  3. Errors may pass silently only because JS sucks.
  4. Eventually is better than never.
  5. If the implementation is hard to explain, it's probably JS's fault.
  6. If the implementation is easy to explain, someone might eventually decide to help me.
  7. Abstract games are one hoking great idea -- let's do more of those.
